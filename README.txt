BlinkenBallz is an Arduino-based project for an LED illumanated ball.

BrightBall is the simplest source code, and was used for the original video.
The BrightBall code is in the public domain.
http://www.youtube.com/watch?v=IUhEINPraN8

The rest is a work in progress.
